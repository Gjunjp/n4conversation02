文法ぶんぽう 1

〜んです

( 說明狀況或理由 )

理由や状況を自然に説明する時に使います。

動詞普通形 / い形 ＋ んです
な形 / 名詞 ＋ なんです
(用於自然地說明理由或目前的狀況。比純粹的「です/ます」更有語氣。)

理由を説明せつめいしましょう！ (來說明理由吧！)

「～んです」を使って答えてください。

Q1: どうして遅刻したんですか？ (為什麼遲到了？)

➔ 電車でんしゃが遅おくれた（　　　）。

Q2: どんな部屋を探していますか？ (在找什麼樣的房間？)

➔ 安やすい部屋へやがいいです。学生がくせい（　　　）。

文法ぶんぽう 2

〜ばいいですか

( 我該怎麼做才好呢？ )

アドバイスや方法を聞く時に使います。

「どうしたらいいですか」と同じ意味です。
(用來詢問建議或方法。遇到困難尋求指示時必備。)

動詞どうし ば形 ＋ いいですか

例れい：どこで住所じゅうしょを登録とうろくすればいいですか。

文ぶんを作つくりましょう！ (來造句吧！)

遇到以下狀況，請試著用「～ばいいですか」來求救。

  • 書き方がわかりません。 (不知道怎麼寫)
    ➔ どう                               
  • どの窓口に行くかわかりません。 (不知道要去哪個櫃台)
    ➔ どの窓口に                               

文ぶんを作つくりましょう！ 解答

遇到以下狀況，請試著用「～ばいいですか」來求救。

  • 書き方がわかりません。 (不知道怎麼寫)
    ➔ どう 書かけばいいですか。
  • どの窓口に行くかわかりません。 (不知道要去哪個櫃台)
    ➔ どの窓口に 行いけばいいですか。
